# React-Home-Page

- File types js + jsx + scss.
- Folder structure:
  - src/images for dynamic images (dummy data images) .
  - src/components for other react components .
  - src/pages for router pages .
  - src/layout for general components "Navbar - Footer" .

---

- Used lazy loading.
